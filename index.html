NukeBot

The easy solution for clean channels

Features

Coded in Java

The bot is completely coded in Java utilizing the Discord JDA API and the top.gg API for updating the server count on their page.

Free forever

The bot will stay completely free forever.

Safe

The bot doesn't collect any data except for the amount of servers it is in.

Active support

I provide different possibilities to contact me, either via Discord or using Twitter to contact the support account provided on this page.

History

22 April 2021 - NukeBot verified by Discord

Today NukeBot actually got verified by Discord, which now took over 4 weeks, but it's finally done and everyone will be able to profit from the channel nuking feature and the tempchannel feature. Additionally, I changed the profile picture because the old one was just stolen from the internet, so I thought it would be time for a custom one. But without you I wouldn't stay here so I want to say thanks for your support!

15 April 2021 - Tempchannels released

Since it takes so long for Discord to verify the bot, I decided to release the tempchannels feature. That means that you can configure a voice channel as a creator for temporary channels so when a user joins it, the bot will create a temporary voice channel for him so you won't need to create many voice channels to meet your community's needs.

5 April 2021 - NukeBot on Twitter

Today I created a Twitter profile for NukeBot on which I'll also be posting updates and providing support. If you want to stay up-to-date, just follow @NukeBotSupport on Twitter or click the "Twitter Support" button on top of this page.

24 March 2021 - Limit reached

My bot now reached the limit of 100 servers and is now waiting for verification. I hope, Discord gets this done really fast

23 March 2021 - Milestone of 75

The bot already reached the 75 servers which means, that I received a message from Discord that I am now able to apply my bot for verification so it'll be able to join more than 100 servers. Also this website got released today.

19 March 2021 - NukeBot approved by top.gg

This morning, I launched Discord as usual, and when I saw that message from top.gg's bot about the approval, I was so happy.
